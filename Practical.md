## Algorithm

## Theory

### Introduction
- Data preprocessing is an important step in Machine Learning.
- Raw data may contain missing values, duplicate data and noise.
- Preprocessing improves the quality and accuracy of dataset.
- Clean data helps in better Machine Learning model performance.

### Handling Missing Values
- Missing values occur when some data entries are empty or null.
- Missing values can reduce model accuracy.
- They can be handled using mean, median or mode methods.
- Rows or columns containing missing data can also be removed.

### Removing Duplicate Data
- Duplicate data means repeated records in dataset.
- Duplicate values increase redundancy and affect analysis.
- Removing duplicates improves data consistency and efficiency.

### Normalization
- Normalization scales data into a fixed range between 0 and 1.
- It helps when features have different ranges of values.
- Normalization improves Machine Learning model performance.

### Standardization
- Standardization converts data into standard form with mean 0 and standard deviation 1.
- It helps in improving accuracy of Machine Learning algorithms.
- It is commonly used in regression and classification problems.

### Categorical Data Conversion
- Machine Learning algorithms work with numerical data only.
- Categorical values are converted into numerical form using encoding techniques.
- Label Encoding and One-Hot Encoding are commonly used methods.

### Outlier Detection
- Outliers are abnormal or extreme values in dataset.
- Outliers can affect prediction accuracy of Machine Learning models.
- They can be detected using statistical methods and visualization techniques.

### Importance of Data Preprocessing
- Improves data quality and consistency.
- Reduces errors and noise in dataset.
- Enhances performance of Machine Learning models.
- Makes dataset suitable for analysis and prediction.

## Conclusion
Successfully studied and performed data preprocessing techniques using Python. Learned handling of missing values, duplicate data, normalization, standardization, categorical data conversion and outlier detection. Data preprocessing improves dataset quality and increases Machine Learning model performance.

## Theory

### Introduction
- Exploratory Data Analysis (EDA) is the process of analyzing datasets before applying Machine Learning algorithms.
- It helps in understanding structure, patterns and relationships in data.
- EDA improves decision making and model performance.
- It is an important step in Data Science and Machine Learning.

### Statistical Measures
- Statistical measures help summarize and analyze data.
- Mean represents average value of dataset.
- Median represents middle value of dataset.
- Mode represents most frequently occurring value.
- Standard deviation measures spread of data from mean.
- Variance measures variability in dataset.

### Correlation Analysis
- Correlation shows relationship between two variables.
- Positive correlation means both variables increase together.
- Negative correlation means one variable increases while the other decreases.
- Correlation coefficient values range from -1 to +1.
- Correlation analysis helps identify important features in dataset.

### Heat Maps
- Heat maps are graphical representations of correlation values.
- Different colors indicate strength of relationships between variables.
- Heat maps help in easy interpretation of large datasets.
- They are useful for identifying highly related features.

### Visualization Plots
- Visualization helps in understanding trends and distributions in data.
- Histogram shows frequency distribution of values.
- Bar chart compares categorical data.
- Scatter plot shows relationship between two variables.
- Line graph represents changes over time.
- Visualization makes data analysis easier and more effective.

### Importance of EDA
- Helps detect missing values and outliers.
- Improves understanding of dataset characteristics.
- Helps in feature selection for Machine Learning models.
- Reduces errors before model training.
- Enhances accuracy and performance of predictive models.

## Conclusion
Successfully studied and performed Exploratory Data Analysis using statistical measures, correlation analysis, heat maps and visualization plots. Learned techniques for analyzing, visualizing and interpreting datasets, which are important for Machine Learning and Data Science applications.
